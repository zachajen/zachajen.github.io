Python-Based Database Assistant

For anyone new to learning SQL, it might be intimidating to memorize the SQL terms and tools, especially if you're working in the shell of MySQL with no GUI.

Luckily, this is the tool for you! Being a Python based user-input program, you tell it what you want to do, and it'll spit back out raw SQL for you to use!

More than that, if you're just wanting to practice or brush up, it has a quiz mode, and it has a dummy data mode (which creates dummy data to fill spots in a practice database).
